# Crear sistemas de diseño accesibles con IA

Monorepositorio base para el taller, creado con Turborepo, PNPM y TypeScript.

## Estructura

- `packages/design-system`: paquete React preparado para publicar.
- `apps/docs`: documentación del sistema de diseño con Storybook.

## Scripts

```bash
pnpm install
pnpm build
pnpm typecheck
pnpm storybook
```

For a fresh Codex worktree, run:

```bash
pnpm setup:worktree
```
